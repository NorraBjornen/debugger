Описание
    Данное приложение является минимальной версией отладчика c графическим интерфейсом для языка программирования python. Эта программа
    позволяет открывать указанный скрипт в некотором подобии текстового редактора, устанавливать брейкпоинты, снимать брейкпоинты и производить отладку:


Требования
    Python версии 3


Состав
    Приложение, выполняющее отладку - interface_debugger.py
    Вспомогательный класс, отвечающий за обработку нажатий на текст - hyperlink_manager.py
    Класс, реализующий логику отладки (упрощенный аналог pdb) - tracer.py
    Приложение, выполняющее роль простого примера кода, который можно отлаживать - script_to_debug.py


Использование приложения

    1. Чтобы установить брейкпоинт нужно нажать на номер строки, соответствующей строке, перед которой пользователь хочет выполнить паузу при исполнении скрипта.
    2. Чтобы снять брейкпоинт нужно выполнить аналогичные дейтвия.
    3. Индикатором того, что брейкпоинт установлен является изменение цвета номера строки, на которой установлен брейкпоинт.
    4. После того, как все брейкпоинты установлены, следует нажать кнопку "Отладить", после чего в консоли можно вводить названия переменных, информацию о которых пользователь
    желает получить. Если потребности выводить переменные больше нет - то при запросе отладчика на название переменной на отображение следует ничего не вводить
    и нажать Enter.
    5. После завершения работы непосредственно процесса отладки (в консоли), можно вернуться к графическому интерфейсу и расставить брейкпоинты по-другому

    Справка по запуску: python debugger.py --help
                            или
                        python debugger.py -h

    Команды
        --py или -p    -   указание пути к файлу, который необходимо отладить

        Пример
            python debugger.py --py script_to_debug.py
            python debugger.py -p script_to_debug.py


    Примеры запуска приложения
        python interface_debugger.py -p script_to_debug.py
