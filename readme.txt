Описание
    Данное приложение является минимальной консольной версией отладчика для языка программирования python. Эта программа написана для получения балла в системе Anytask,
    что является необходимым требованием для исключения исхода событий, при которых задача перестанет быть закрепленной за автором


Требования
    Python версии 3


Состав
    Приложение, выполняющее отладка - debugger.py
    Приложение, выполняющее роль простого примера кода, который можно отлаживать - script_to_debug.py


Тесты
    debugger_test.py


Использование приложения

    Справка по запуску: python debugger.py --help
                            или
                        python debugger.py -h

    Команды
        --py или -p    -   указание пути к файлу, который необходимо отладить

        Пример
            python debugger.py --py script_to_debug.py
            python debugger.py -p script_to_debug.py
        -------------------------------------------------

        --bp или -b    -   указание набока точек остановки для отладки кода и просмотра переменных

        Пример
            python debugger.py --bp 1,2
            python debugger.py -b 2,4,6,7,5

        Указание этого параметра не является обязательным

    Примеры запуска приложения
        python debugger.py -p script_to_debug.py -b 2,4,6
